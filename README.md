# SocalASA Poster Competition
To replicate the results reported, upload the dataset 'Sephora Kids Linkedin (Responses).xlsx' into Google Colab and run the code 'sephora_baysian_v2_asa.ipynb'

## Notes
- Computational steps and code are adapted from *Bayes Rules!* by Alicia A. Johnson, Miles Q. Ott, and Mine Dogucu (University of California, Irvine).
- Prompts for gemini also followed the steps from the Books. Book provided R Code.
- Python Code for analysis was generated using Gemini. file name: sephora_baysian_v2_asa.ipynb
- Survey administered (google form) included in the folder. Filename:survey_sephora.png
- Data collected : Sephora Kids Linkedin (Responses).xlsx . Note: this data collection was part of an earlier study that was conducted in 2025 December.
